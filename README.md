# Deaf Communicate

Simple iOS/iPadOS app to help communicate with those who are hard of hearing. Uses Apple's speech framework to convert spoken audio into text.

## Features
- Use the keyboard to type what you want to say
- Auto text scroll makes the text automatically scroll as speech is being converted to text
- Dictate using the microphone button
- Save text into the text history to reference it later
- Copy what you said with the copy button
- Share what you said with the share button
- Clear the text through one quick tap on the trash can button
- Look at all the text you have saved in your text history and tap on one to restore it
- Adjust font size, style, and color via settings page
- Enable/disable auto text scroll via settings page
- Supports both light and dark mode

## Requirements
iOS/iPadOS 17+
